# urxvt extensions
## defaults
* defaults - urxvt's default extensions  
	http://cvs.schmorp.de/rxvt-unicode/src/perl/
## autocomplete-ALL-the-things
* autocomplete-ALL-the-things - Arbitrary text completion for urxvt  
	https://github.com/Vifon/autocomplete-ALL-the-things
## clipboard, keyboard-select, url-select
* clipboard - Use keyboard shortcuts to copy the selection to the clipboard and to paste the clipboard contents (optionally escaping all special characters).  
* keyboard-select - Use keyboard shortcuts to select and copy text.  
* url-select - Use keyboard shortcuts to select URLs.  
	https://github.com/muennich/urxvt-perls
## tabbedex
* tabbedex - Tabbed plugin for rxvt-unicode with many enhancements  
	https://github.com/stepb/urxvt-tabbedex
## tabbedex (mina86)
* tabbedex (mina86) - Tabbed plugin for rxvt-unicode with many enhancements  
	https://github.com/mina86/urxvt-tabbedex
## cdmn
* cdmn - cdmn (cpu, disk, memory, network) is a Perl extension for urxvt which extends urxvt to show the utilization of different system resources  
	https://github.com/Jeansen/cdmn
## config-reload
* config-reload - This plugin allows you to reload urxvt configuration at runtime by sending SIGHUP to the urxvt process  
	https://github.com/regnarg/urxvt-config-reload

* urxvt-pcon - URXVT perl console plugin

## font-size, fullscreen, keyboard-select, vtwheel
* font-size - On-the-fly adjusting of the font size in urxvt  
* fullscreen - Toggle fullscreen

* vtwheel - emulate scroll wheel for secondary screen
